1. Document Object Model (DOM), JavaScript can change all the HTML elements, attributes and CSS Style in the page. JavaScript can remove and add existing HTML elements and attributes.

2. Javascript DOM is most commonly used to get or modify the content or value of the HTML elements on the page.
getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

3. Event bubbles are a concept in the JavaScript DOM. When an event occurs on another element, it first fires handlers, then its parent, then other ancestors.

4.callback function is a function that is passed as an argument to another function, to be called later when needed. which is then called inside the outer function to perform some sort of task. The purpose of callbacks is to execute code in response to an event.
